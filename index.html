This page holds no data. It loads your postings live from your private repo using a token you provide, kept only in this browser.

Create a fine-grained token (one-time):

  1. Open GitHub → Fine-grained tokens → Generate new token.
  2. Name it (e.g. job-dashboard) and pick an expiration.
  3. Resource owner: your account. Repository access: Only select repositories → choose .
  4. Permissions → Repository permissions → Contents: Read and write. (Metadata read is added automatically.) Nothing else.
  5. Generate, copy the github_pat_… value, and paste it below.
